Start the Remotion Studio dev server for the video project.

Run the following command in the background and let the user know it's starting:

```bash
cd /home/user/claude-ads/video && npm run dev
```

Inform the user that Remotion Studio is starting and will be available at http://localhost:3000 (or the port shown in the output). Tell them to open that URL in their browser to preview and edit their video compositions.
