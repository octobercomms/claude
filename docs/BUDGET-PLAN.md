# nvelope.co — Budget Plan
**Updated:** March 2026

---

## Budget Models

nvelope.co has two distinct budget scenarios:

### Model A — Platform Pays (nvelope runs ads on behalf of clients)
nvelope funds ads, recoups cost via client subscription / per-lead fee

### Model B — Studio Pays (each studio funds their own campaign)
Each studio pays for their own ad budget; nvelope manages the accounts

> Both models use the same campaign architecture. The difference is billing and reporting.

## Budget Distribution Within Each Platform

### Google — Internal Split
```
55% → Non-Brand Search (high-intent, primary leads driver)
25% → Brand Search (protect brand, capture direct intent)
10% → Remarketing Display (quiz abandoners, page visitors)
10% → Performance Max (Month 3+ only, after conversion data exists)
```

### Meta — Internal Split
```
50% → Prospecting (Advantage+ broad targeting)
30% → Retargeting (visitors + quiz starters)
20% → Lookalike audiences (activate at 100+ leads)
```

### X — Internal Split
```
60% → Interest targeting (architecture + home renovation)
25% → Follower lookalike targeting (design publication audiences)
15% → Keyword/conversation targeting
```

## Scaling Decision Framework

### When to Scale Up (20% Rule)
```
IF actual CPL < target CPL by more than 10%
AND you have ≥15 leads in last 30 days (Google) / ≥50 conv in 7 days (Meta ad set)
AND campaign is NOT in learning phase
THEN increase budget by 20% max
→ Wait 3–5 days, check CPL impact before next increase
→ Never exceed 20% increase in one step on Meta (resets learning)
```

### When to Pause (3× Kill Rule)
```
IF spend > 3× target CPL
AND leads = 0
THEN pause that ad / ad set / campaign
→ Diagnose: creative? targeting? landing page? tracking?
→ Do not reactivate without changes
```

### When to Reduce
```
IF CPL increased >15% after a budget increase
THEN roll back to prior budget level
→ Wait 7 days
→ Try horizontal scale instead: new city, new studio, new creative
```

## Per-Studio Billing Options

| Model | How It Works | Best For |
|-------|-------------|----------|
| Flat retainer | Studio pays $X/month; nvelope manages all ads | Predictable revenue |
| % of ad spend | Studio pays media cost + 15–20% management fee | Scales with studio ambition |
| Per lead | Studio pays $X per qualified lead delivered | Risk-share; good for early trust |
| Hybrid | Retainer + per-lead above threshold | Most common agency model |

## Saturation Signals — When to Expand vs Refresh

| Platform | Signal | Threshold | Action |
|----------|--------|-----------|--------|
| Google | Impression Share | >80% | Expand to new keywords or cities |
| Meta | Frequency (7-day) | >4 | Refresh creative; expand audience |
| X | CTR declining | >20% drop over 14 days | New creative or new audience |
| All | CPL rising | >15% MoM | Diagnose and fix before scaling |
